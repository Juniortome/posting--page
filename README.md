
Posting Page


Projeto front-end que simula a criação de um post no estilo LinkedIn, permitindo ao usuário inserir um título e um conteúdo, enviar esses dados via JavaScript (fetch) para uma API aberta e renderizar o retorno na própria página.



## Objetivo
Consolidar fundamentos essenciais do front-end: HTML semântico, CSS básico e JavaScript com uso de API.

## Características
O usuário preenche o formulário com título e conteúdo, envia o post e vê o retorno renderizado dinamicamente na tela.
